How to Migrate a Dashboard

The create-from-tableau skill in Vizion  Full guide

Setup
0
Connect Tableau MCP
Add your personal token to agent config. Go to tableau.wixpress.com → Account → Settings → Personal Access Tokens → Create Token
Then add to your mcp.json:
"Tableau-MCP": {
  "url": "https://bo.wix.com/_serverless/tableau-mcp/mcp",
  "env": {
    "TABLEAU_SERVER_URL": "https://tableau.wixpress.com",
    "TABLEAU_TOKEN_NAME": "your-token-name",
    "TABLEAU_TOKEN_KEY": "your-token-key"
  }
}
1
Start the migration
Ask your coding agent to migrate a dashboard, or call /create-from-tableau
2
Link the dashboard
Paste the Tableau URL, or give the dashboard name - the agent will search for it
3
Set target location in Vizion
State where the new dashboard should live in Vizion
Build
4
Review the plan
The agent creates a .md plan file with the migration blueprint - review and make changes before building
5
Build the new dashboard
When ready, tell the agent to build - it generates the Vizion dashboard
Verify & Ship
6
Data & UI verification
Continue to the data and UI verification stages - compare with the original
7
Review locally
Review the new dashboard yourself in your local environment
8
Publish
Ship it! Update the migration tracker sheet with the Vizion URL
No installation needed - the skill is already in Vizion. Just connect your Tableau token and go.
